Como usar los programas de la pregunta 4

1 Tener instalado Ruby en su ordenador y el complemento "gem install simplecov" .

2 En terminal en la ruta de la termina poner "ruby main.rb"  o "ruby test_vector3d.rb"
segun se quieran usar y dar enter, el programa se ejecutara sin problemas.

3 El main es el programa solicitado mientras que test_vector3d es la cobertura
que se solicita en el ejercicio. vector3d.rb es un requerimiento necesario para
que el main sirva.

Ejemplo de uso:
Como al ejecutar el programa se arrojan los resultados directamente, debera cambiar los valores
o las opereciones que quiera hacer directamente en el main en esta seccion 

puts "b + c = #{b + c}"
puts "a * b + c = #{a * b + c}"
puts "(b + b) * (c - a) = #{(b + b) * (c - a)}"
puts "a % (c * b) = #{a % (c * b)}"
puts "b + 3 = #{b + 3}"
puts "a * 3.0 + a.norm = #{a * 3.0 + a.norm}"  
puts "(b + b) * (c % a) = #{(b + b) * (c % a)}"
